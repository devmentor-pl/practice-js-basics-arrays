# `#01` JavaScript: Podstawy – tablice

Następnie wyświetl w konsoli informacje o pierwszym, trzecim i piątym użytkowniku. Dodatkowo z pomocą właściwości `.length` wyświetl, ile elementów zawiera tablica.
